⚙ Configuración de Google Drive
Solo se hace una vez. 5 minutos.
1
Ir a console.cloud.google.com y crear un proyecto nuevo.
2
En el menú: APIs & Services → Library. Buscar y habilitar Google Drive API.
3
APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID. Tipo: Web application.
4
En Authorized JavaScript origins agregar la URL donde abrís este archivo (ej: http://localhost o el dominio donde lo hosteés).
